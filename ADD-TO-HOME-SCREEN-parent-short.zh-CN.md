# 家长转发版：将网页添加到平板桌面

请家长帮孩子先操作一次，后续孩子直接点桌面图标进入即可。

## iPad

1. 用 `Safari` 打开老师发的网址。
2. 点击右上角 `分享`。
3. 选择 `添加到主屏幕`。
4. 点击 `添加`。

## Android 平板

1. 用 `Chrome` 打开老师发的网址。
2. 点击右上角 `...`。
3. 选择 `添加到主屏幕` 或 `安装应用`。
4. 点击 `添加` 或 `安装`。

## 温馨提示

- 建议把图标拖到桌面首页，方便孩子找到。
- 如果图标打不开，先确认网络正常，再重新进入一次。
- 如果需要，也可以删除后重新添加。
