# 将网页添加到平板桌面

适用场景：把当前教学网页添加到平板桌面，方便幼儿像打开 App 一样一键进入。

建议由老师或家长先完成一次添加，再交给幼儿使用。

## iPad（推荐使用 Safari）

1. 在 `Safari` 中打开老师提供的网址。
2. 等页面完全打开后，点击右上角的 `分享` 按钮。
3. 在弹出的菜单中，点击 `更多`，再点击 `添加到主屏幕`。
4. 如果页面出现 `作为网页 App 打开` 或 `Open as Web App`，建议保持开启。
5. 点击右上角 `添加`。
6. 返回桌面后，就会看到这个网页的图标。

使用建议：
- 第一次添加后，可以把图标拖到桌面第一页，方便幼儿找到。
- 如果需要，也可以把图标拖到底部 Dock，打开会更快。

## Android 平板（推荐使用 Chrome）

1. 在 `Chrome` 中打开老师提供的网址。
2. 等页面完全打开后，点击右上角 `...` 菜单。
3. 找到并点击 `添加到主屏幕`。
4. 如果看到的是 `安装应用` 或 `Install`，优先选择这个选项。
5. 按提示点击 `安装` 或 `添加`。
6. 返回桌面后，就会看到这个网页的图标。

使用建议：
- 如果系统询问是否允许添加到主屏幕，请选择 `允许`。
- 添加完成后，可以长按图标并拖动到桌面首页。

## 常见问题

### 1. 为什么 iPad 上找不到“添加到主屏幕”？

- 请确认使用的是 `Safari`，不是其他浏览器。
- 可以先点 `分享`，再向下滑动操作列表查找 `添加到主屏幕`。

### 2. 为什么 Android 上没有“安装”按钮？

- 有些设备显示的是 `添加到主屏幕`，有些会显示 `安装应用`，两种都可以。
- 建议优先使用 `Chrome` 打开网页。

### 3. 添加后打不开怎么办？

- 先确认平板已经联网。
- 关闭网页后重新从桌面图标进入一次。
- 如果还是不行，可以删除桌面图标后重新添加。

### 4. 删除桌面图标会不会影响网页内容？

- 不会。删除的只是桌面快捷方式，不会删除网站本身。

## 给老师的简短提示

建议首次由成人完成以下操作：

- 打开网页
- 添加到主屏幕
- 把图标移动到桌面首页
- 让幼儿以后只点击桌面图标进入

这样可以减少误操作，也更适合课堂使用。
